NAO Robot Research Project

The NAO Robot Research Project was a collaboration between John Marble and the Collabortive Robotics and Intelligent Sytems (CRoIS) Lab at The College of New Jersey. The first project was started and completed in the Summer of 2022 during Mentored Undergraduate Research Experience (MUSE), with the primary goal of implementing emotion recognition algorithms on the NAO robot.

The project involved programming the robot using Python and integrating it with various sensors and other hardware components. John Marble was responsible for writing the majority of the code and working with various NAO robot SDKs and APIs available from the parent company, Softbank Robotics, in order to sucessfully recognize up to six different human emotions.

As the project was a success, data collected during each experimental trial was written to an excel sheet that saved the predicted emotion, an image of the subject, and various values relating to key facial features that aided in the emotion classification process.
