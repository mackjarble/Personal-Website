Hack-Io-thon Weather Application

Project Description

This project was developed as part of a hackathon event where we had to build a weather application that could display weather information for different sensors attached to the Particle Boron. The project was developed using C++ and MATLAB for an IoT application. The application allows users to visualize the weather conditions of the location where the system is placed using MATLAB graphs that are live updated to a web presentation.

Features

  • Responsive design
  • Displays current values for temperature, pressure, humidity, and illuminance
  • Displays live graphical data from sensors

Technologies Used

  • Particle Boron Microcontroller
  • C++
  • Bread board prototyping
  • I2C sensor implementation
  • Webhooks
  • MATLAB
  • Live data web presentation
