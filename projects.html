  • Hack-Io-Thon Weather application

A feature of this event was the hack-io-thon competition, where teams competed to provide a solution to an IoT application involving a microcontroller and multiple peripheral sensors to gather weather information about the environment and then present that information using a web presentation.

  • Swim Timer Repair Project

    This project involved the repair of a nonfunctional timing system that was used fifty years ago. Upon inspection, all of the circuits and components were greatly damaged from residing in the harsh aquatic environment. The solenoid driven sevent segment timing display was brought back to life when the internal circuitry was re-engineered and controlled using an Arduino MEGA microcontroller that managed and coordinated relay functionality and display information.
